# ugly-beauty-desktop
主要以vue進行開發，完成一個模擬桌面的頁面。

## 網站功能
- 使用者可以點擊桌面圖式開啟指定視窗。
- 使用者可以拖曳視窗。
- 使用者可以播放、暫停、停止播放網頁內播放器的音樂。
- 使用者可以切換網頁內音樂播放器的音樂。
- 使用者可以透過切換桌面的視窗變更桌面。
